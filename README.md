# RestAPI Starter
Simple Starter Project that helps to build Rest API and Back End Server, Write in NodeJS.

## Installation
- Use the package manager NPM to install.

```javascript
npm install
```

## Usage
```javascript
npm start
```

## Develop
Open index.js and start developing.
